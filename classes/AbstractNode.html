AbstractNode Class

Defined in: pedigree/abstractNode.js:1

AbstractNode is the general abstract class for nodes on the Pedigree graph. An AbstractNode contains information about its position on the canvas and about relationships with other nodes on the graph.

isRelatedTo

(
  • nodeID
  • [visited=[]
)
Array

Defined in pedigree/abstractNode.js:386

Returns true if this node is related to the node with the id nodeID

Parameters:

  • nodeID Number

    ID of any node in the graph

  • [visited=[] Array optional

    A list of IDs of nodes that shouldn't be considered

Returns:

Array: An array in the form of [result, visitedNodes]

Example:

var malePerson = editor.getGraph().addPerson(100,100, "M", 20);
var femalePerson = editor.getGraph().addPerson(300,100, "F", 21);

malePerson.isRelatedTo(femalePerson.getID()) // -> [false, [20]]

Breaks connections with related nodes and removes the node and its visuals.

Parameters:

    if true, will remove all nodes that will end up disconnected from the Proband

  • [skipConfirmation=false] Boolean optional

    if true, no confirmation box will pop up

           PregnancyNodes : [Pregnancy1, Pregnancy2, ...],
           PersonGroupNodes : [PersonGroup1, PersonGroup2, ...],
           PlaceHolderNodes : [PlaceHolder1, PlaceHolder2, ...]
       },
       created: [PlaceHolder1, PlaceHolder2, ...]
    }
