AbstractNodeVisuals Class

Defined in: pedigree/abstractNodeVisuals.js:1

AbstractNodeVisuals is the general abstract class for the graphic engine used by nodes on the Pedigree graph. Objects of this class have information about the positioning of the graphical elements of the node.
