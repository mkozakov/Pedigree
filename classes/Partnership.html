Partnership Class

Extends AbstractNode
Defined in: pedigree/partnership.js:1

Partnership is a class that represents the relationship between two AbstractNodes and their children.

Defined in pedigree/partnership.js:1

Parameters:

  • x Object

    the x coordinate at which the partnership junction will be placed

  • y Object

    the y coordinate at which the partnership junction will be placed

  • partner1 Object

    an AbstractPerson who's one of the partners in the relationship.

  • partner2 Object

    an AbstractPerson who's the other partner in the relationship. The order of partners is irrelevant.

  • nodeID
  • [visited=[]
)
Array

Inherited from AbstractNode: pedigree/abstractNode.js:386

Returns true if this node is related to the node with the id nodeID

Parameters:

  • nodeID Number

    ID of any node in the graph

  • [visited=[] Array optional

    A list of IDs of nodes that shouldn't be considered

Returns:

Array: An array in the form of [result, visitedNodes]

Example:

var malePerson = editor.getGraph().addPerson(100,100, "M", 20);
var femalePerson = editor.getGraph().addPerson(300,100, "F", 21);

malePerson.isRelatedTo(femalePerson.getID()) // -> [false, [20]]
