Defined in pedigree/pedigree.js:307

Find the best position to insert one or more new neighbors for an existing node

Parameters:

  • relativePosition Object

    an object with one field, which can be either 'above', 'below', 'side', or 'join', whose value indicated the id of a node

  • identifiers Object

    an array of new ids for which positions must be found

Returns:

an object where each field is one of the ids given as input, and the value is the point where that node should be placed
