PersonHoverbox Class

Extends AbstractHoverbox
Defined in: pedigree/personHoverbox.js:1

PersonHoverbox is a class for all the UI elements and graphics surrounding a Person node and its labels. This includes the box that appears around the node when it's hovered by a mouse, as well as the handles used for creating connections and creating new nodes.
