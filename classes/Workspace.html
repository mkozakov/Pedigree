Workspace Class

Defined in: pedigree/workspace.js:1

Workspace contains the Raphael canvas, the zoom/pan controls and the menu bar on the top. The class includes functions for managing the Raphael paper object and coordinate transformation methods for taking pan and zoom levels into account.
